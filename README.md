# Forensics-Investigation

This repository documents a digital forensic investigation involving both **disk image analysis** and **network traffic analysis**.  
The objective was to identify evidence of **identity theft, credit card fraud, and malicious network activity** using forensic methodologies.

The investigation consists of two parts:

1. Disk Image Forensics
2. Network Traffic Forensics

---

# Lab Environment

## Tools Used

- Autopsy
- FTK (Forensic Toolkit)
- Wireshark

---

# Key Skills Demonstrated

- Disk Image Forensics
- Evidence Integrity Verification
- Metadata Analysis
- Hidden Data Discovery
- Network Packet Analysis
- Port Scanning Detection
- Telnet Session Investigation
- SQL Injection Analysis
- Data Exfiltration Detection
